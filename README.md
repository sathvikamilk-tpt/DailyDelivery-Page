# Milk Delivery Manager

Single-file, local-first milk delivery app.

## Files
- `index.html` — the app

## Run locally
Use any static server:
```bash
python -m http.server 8000
```
Then open `http://localhost:8000`.

## Free hosting
- GitHub Pages
- Netlify
- Vercel static deployment

## Recovery
Use **Export JSON** regularly and keep the file on Google Drive or in Files.
